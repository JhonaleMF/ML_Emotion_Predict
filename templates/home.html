Modelo detección de emociones de un Tweet

Esta API puedes usarla de dos maneras, a través de esta interfaz o atacando los endpoints correspondientes usando las URL's respectivas, en este apartado podrás ver las URL's y el formato necesario para hacer la consulta.

Puedes usar esta interfaz gráfica para dirigirte al endpoint de la detección de emoción "Consulta", allí, ingresar el texto y dar clic en "submit". Posteriormente, recibirás la emoción correspondiente al texto introducido, con su respectiva información.

Consulta

También puedes hacer solicitudes mediante URL con los siguientes endpoints (parametro o key "text", con su respectivo tweet(value)):

- /api/v1/consulta?text=Respuesta ingresada
